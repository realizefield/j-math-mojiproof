文字式の証明 ／ 中学2年　教科書掲載パターン
例題
📌 このページで学ぶこと 3桁の整数 100a + 10b + c を変形して，
99a + 9b + (a + b + c) の形をつくることが核心です。

✅ 3の倍数判定：a + b + c が 3 の倍数 → 整数も 3 の倍数
✅ 9の倍数判定：a + b + c が 9 の倍数 → 整数も 9 の倍数

🔑 ポイント：100 = 99 + 1，10 = 9 + 1 と分解するのが変形の鍵です！
3桁の整数の各位の数の和が 3 の倍数ならば，その整数も 3 の倍数であることを証明しなさい。

STEP 1　3桁の整数を文字式で表しましょう

百の位を a，十の位を b，一の位を c とおくと，
3桁の整数は 100a + 10b + c と表せます。

各位の数の和は a + b + c です。

STEP 2　100 と 10 を分解しましょう ここが核心です！

100 = 99 + 1，10 = 9 + 1 と分解して，
100a + 10b + c を変形します。

100a + 10b + c
= (99 + 1)a + (9 + 1)b + c
= 99a + a + 9b + b + c
= 99a + 9b + (a + b + c)

これで「各位の和 a + b + c」が現れました！

STEP 3　3 の倍数であることを示しましょう

99a + 9b + (a + b + c) を見ると，

・ 99a = 3 × 33a → 3 の倍数
・ 9b = 3 × 3b　→ 3 の倍数
・ a + b + c　　→ 3 の倍数（条件より）

3つすべてが 3 の倍数ですから，和も 3 の倍数です。

99a + 9b + (a + b + c) = 3(33a + 3b) + (a + b + c)

a + b + c が 3 の倍数のとき，全体は 3 の倍数です。

【証明】
百の位を a，十の位を b，一の位を c とおく。
3桁の整数は 100a + 10b + c と表せる。

100a + 10b + c
= (99 + 1)a + (9 + 1)b + c
= 99a + 9b + (a + b + c)
= 3(33a + 3b) + (a + b + c)

3(33a + 3b) は 3 の倍数であり，a + b + c も 3 の倍数だから，
100a + 10b + c は 3 の倍数である。　■

STEP 4　9の倍数判定への応用

同じ変形で 9 の倍数判定もできます。

99a + 9b + (a + b + c) = 9(11a + b) + (a + b + c)

9(11a + b) は 9 の倍数ですから，
a + b + c が 9 の倍数ならば，全体も 9 の倍数になります。

✅ まとめ：変形の核心は
「100a + 10b + c = 99a + 9b + (a + b + c)」
この式さえ導けれれば，3・9 どちらの証明もできます！
